## [0.0.1] - 2024-03-14

### 新增

- 为 KMScript (.km) 文件提供基本的语法高亮支持
- 实现了关键字、函数、字符串、注释、数字和操作符的高亮
- 添加了基本的语言配置，包括注释和括号匹配
- 创建了初始的 README 文件，提供扩展的基本信息和使用说明

### 已知问题

- 功能还比较基础，仅提供基本的语法支持

### 未来计划

- 完善代码片段（snippet）功能
- 实现代码格式化
- 为函数添加符号
- 为所有函数和关键词添加自动补全功能
- 为所有函数补充参数数量信息

我们期待在未来的版本中为您带来更多功能和改进！
